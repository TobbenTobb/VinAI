Para Runear el programa se nececita

Nececitas instalar

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
pip install rasa 

Es el motor principal del chatbot. 
Al instalar rasa, automáticamente se instalan también rasa-sdk, tensorflow, sanic y muchas otras dependencias que el bot necesita para correr.

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

pip install mysql-connector-python

Esta es la librería clave que permite a Python conectarse a tu base de datos MySQL. 
Es usada tanto por actions.py (para guardar valoraciones y leer vinos) como por admin_app.py (para mostrar el dashboard y el perfil).

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

pip install Flask 

Es el micro-framework web que usas para correr todo tu panel de administrador, incluyendo el login y la página de perfil (todo tu archivo admin_app.py).

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

pip install Flask-Login

Lo importas en admin_app.py para manejar la sesión segura del administrador (el login_required, current_user, etc.).

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

pip install Flask-Cors

Lo usas en admin_app.py (CORS(app)) para permitir que tu chat (en localhost:8000) pueda hacer peticiones a tu servidor de login (en localhost:8080).

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

pip install Werkzeug

Lo usas para el hashing seguro de contraseñas (generate_password_hash, check_password_hash). 
Nota: Esta librería casi siempre se instala automáticamente junto con Flask, pero es bueno saber que es una dependencia.

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Ahora tendras que abrir 4 Terminales distintas 

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Primera terminal

python -m http.server 8000  

Para iniciar el servidor y funcione las paginas

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Segunda Terminal

python admin_app.py 

Para Habilitar el funcionamiento de admin_app

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Tercera Terminal 

rasa run actions --debug 

Para Habilitar El Bot con su pensamiento

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Cuarta Terminal

rasa run --enable-api --cors "*" --debug  

Para Habilitar las funciones del Bot

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Si llegaras a modifciar Cualquier Archivo que termine en .yml Tendras que parar todas las terminales y en una nueva terminal usar

rasa train 

Con esto se entrenara deneuvo el modelo

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Luego vuelves a runear todo en sus terminales
